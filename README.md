## RESTful web service

### 1. 요약

Spring Boot JPA를 이용하여 RESTful Web Services 개발하기 위한 개인 스터디 프로젝트입니다.

### 2. 기술 스펙

| Tech    | Version           |
|---------|-------------------|
| **JDK** | Java 11           |
| **F/W** | Spring Boot 2.5.11 |
| **DB**  | H2                |
| **ORM** | Hibernate         |
